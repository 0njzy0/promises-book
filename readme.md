[http://azu.github.io/promises-book/](http://azu.github.io/promises-book/ "Promises book") から閲覧することが出来ます。

## Installation

Asciidocを使い書かれています。

[Asciidoctor](http://asciidoctor.org/ "Asciidoctor")でビルドすることが出来ます。

``` sh
gem install asciidoctor coderay
npm install -g gulp
npm install
make html
open index.html
```
